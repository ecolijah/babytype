# babytype

Simple tool to help with typing fundamentals.
[Site is Live!](https://babytype.xyz)


## run locally: 

### Project Setup

```sh
git clone https://github.com/ecolijah/babytype.git
cd babytype
```
```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
